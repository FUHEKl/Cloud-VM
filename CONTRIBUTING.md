# Contributing

Thanks for your interest in CloudVM.

## Current contribution policy

This repository is currently published as an **academic showcase** and is **read-only for external contributions**.

- External pull requests are not being accepted at this time.
- Public issues may be limited to clarification/questions only.

## How to request collaboration

## Internal contributor notes

When updates are accepted by maintainers:
- keep commits focused and descriptive,
- avoid committing secrets, certificates, or generated keys,
- update `README.md` and security docs when behavior changes.
